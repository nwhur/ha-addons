# Humicon MQTT Daemon

휴미컨(Humicon) 통합 제어기를 홈어시스턴트(HA)와 연동하기 위한 강력하고 스마트한 하이브리드 MQTT 데몬입니다.

## 주요 기능
* **초고속 반응형 제어:** 룸콘의 RS485 통신 패킷을 실시간으로 스니핑(Sniffing)하여, 물리 조작 시 HA UI에 즉각적으로 상태가 반영됩니다.
* **통합 팬(Fan) 제어 UI 지원:** 홈어시스턴트의 공식 Fan 엔티티를 통해 스마트, 제습 자동/수동, 환기 자동/수동, 청정 자동/수동 및 전원 On/Off를 직관적으로 제어할 수 있습니다.
* **센서 데이터 통합:** 온도, 습도, CO2, 미세먼지(PM10), 초미세먼지(PM2.5) 수치를 실시간으로 HA에 제공합니다.
* **디테일한 습도 설정:** 목표 습도를 30%~60% 구간에서 5% 단위로 정밀하게 설정하거나 에코/쾌적/건조 모드로 선택할 수 있습니다.
* **가용성(Availability) 보장:** 기기 전원 차단이나 통신 단절 시 즉각적으로 기기를 비활성화 처리하고 자동 복구 로직을 수행합니다.

## 설치 및 설정 (사전 준비)
이 애드온을 사용하려면 휴미컨 통합 제어기 RS485 단자에 **Elfin EW11** 기기가 연결되어 있어야 합니다.

1. **EW11 설정:** EW11 관리자 페이지에서 Protocol을 `None`(Transparent) 모드로 설정하고 Local Port(기본: 8899)를 개방합니다.
2. **애드온 설치:** 설치 후 상단의 `Configuration` 탭으로 이동합니다.
3. **설정값 입력:** EW11의 IP 주소와 HA MQTT Broker 정보를 입력하고 저장합니다.
4. **시작:** 애드온을 시작하고 로그를 확인하여 정상 연결되었는지 확인합니다!

자세한 매뉴얼은 `Documentation` 탭을 참조해 주세요.
